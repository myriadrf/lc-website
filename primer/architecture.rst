Network Architecture
====================

.. toctree::
   :maxdepth: 2
   :hidden:

Mobile networks are split at a high level into the RAN, core network and IMS core. The most basic LTE network is comprised of a single eNodeB for the RAN, plus a core network which this connects to. An IMS core would also be required in order to support native voice and SMS, but this is optional and can be omitted if only a data service is required.
